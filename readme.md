Inicial App em Flask
===========

Instalação
------

- rodar comando a baixo para instalar dependecias.

````
pipenv install
````

Rodar Projeto
-----

- para rodar , configirar as dependecias no settings e depois rodar os comandos a baixo.

- migrar o banco de dados.
 
````
 flask db migrate
 flask db upgrade 
````

- subir applicaçao

````
 flask run  
````
